# Go Cargo Logistics — Enterprise Logistics Platform

## Vision
Complete production-ready logistics platform for Go Cargo Logistics (gocargologisticsus.com), rivaling FedEx, DHL, UPS quality. Built for thousands of daily users across customer portal, public website, admin dashboard, tracking system, and operations management.

## Design
Authoritative professional logistics platform with corporate trust signals.

- `--background: 0 0% 100%` (pure white)
- `--foreground: 215 25% 27%` (slate-800 #1E293B)

**Style:** Corporate professional with subtle blue gradients, generous whitespace, modern card layouts, smooth animations. Trust-driven design with precise execution.

## Features
- Public website (15+ pages)
- Shipment tracking with live GPS maps
- Customer portal with authentication
- Admin dashboard (shipments, quotes, customers, analytics)
- Quote request system with workflow
- Document generation (BOL, invoices, labels)
- Blog CMS
- Notification system
- Analytics & reporting
- Role-based access control
- Multi-carrier/driver management
